# Smart Blog API


## Running The App

### Running Locally
Make sure to have Maven 3.9+ and Java 21+ installed.
```bash
./mvnw spring-boot:run
```
Note: Make sure ```src/main/resources/application.yaml``` is configured correctly

### Running With Docker
Make sure to have Docker and Docker Compose installed.
```bash
docker-compose up --build
```

## API Endpoint
Access OpenAPI Swagger on ```http://localhost:8080/swagger-ui.html```
